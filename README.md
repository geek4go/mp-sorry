# mp-sorry
基于 [xtyxtyx/sorry](link1) 的小程序版本

主要目的：上手微信小程序，练手用。

### 前端
---
主页：  
- [x] 选择模板
- [] 根据模板发送生成Gif的请求  
展示页： 
- [] 展示生成的Gif图
- [] 提供下载及分享
列表页：  
- [] 以列表的形式展示过去24小时生成的Gif图
- [] 提供下载及分享


### 后台
---
由于xtyxtyx大神提供的api地址不符合微信小程序的要求（未备案）  
所以只能使用自己的服务器来反向代理大神的链接。  
- [] 用自己的服务器反向代理大神的Api地址
